Background Images

The background and background-image properties add an image (jpg, gif, png, webp, or svg) to the inside of an HTML element. The url() value should include file path and file name of the image you want to use.

  • By default the image starts in the top left corner of the element, but you can change its positioning.
  • The images repeat vertically and horizontally unless you tell them not to.
  • Unlike adding an image with the HTML <img> element, CSS background images do not take up their own space on the page. They are contained within the element's box.

Basic Syntax:

h1 {
  background-image: url(example.jpg);
}
background-repeat:
no-repeat; Won't repeat in any direction
repeat-x; Repeats only horizontally
repeat-y; Repeats only vertically
repeat; Repeat in all directions (default behavior)
background-position:
top;
bottom;
left;
center;
right;
%;
px;
background-size
cover; Resizes background image you you can see ALL of it
contain; Resizes image to be big enough to cover the leighth OR the width, whichever is longest. Some of the image is likely to be cropped.
% % Two percentage values. The first is the width and the second is the height.
px Two pixel values. The first is the width and the second is the height.

Shorthand Syntax:

To shorten the code, you can combine multiple properties into the single background: property. The properties must come in this order:

background: color url() repeat attachment position size;
